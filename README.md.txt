## بازی حدس کلمه

این پروژه یک بازی ساده حدس کلمه است که با زبان جاوا نوشته شده است
کاربر باید با حدس زدن حروف یا حدس کامل کلمه، آن را کامل کند و امتیاز بگیرد

## امکانات
- حدس یک حرف
- حدس کل کلمه بعد از مشخص شدن چند حرف
- سیستم امتیازدهی
- راهنما  یک‌بار مصرف (Hint)

## ابزارهای استفاده شده
- Java (JDK)
- IntelliJ IDEA

## نحوه اجرا
- پروژه را در IntelliJ IDEA باز کنید
- فایل Main.java را اجرا کنید
- دستورات نمایش داده شده در کنسول را دنبال کنید
